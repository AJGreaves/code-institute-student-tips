Newbie Tips

For code institute students

Problem

The CSS on your project is not working properly, or is having no effect at all.

Possible Reason #1

The cache in your browser is not refreshing to your new CSS changes.

How to fix it:

When you view your code, always view it in a separate tab:

  • In cloud 9, make sure your html file is open, click “run” at the top of the page.
  • Click on the link provided at the bottom of the screen and select “Open” (NOT “Open in preview”, the preview window is buggy)

Now that you are viewing your page in a separate tab:

  • Click `Ctrl-f5` on windows or `CMD + SHIFT + R` on mac.
  • This will force the browser to refresh the cache and display your new CSS changes.
Possible Reason #2

You didn’t save your html file and/or your CSS file before refreshing the browser

How to fix it
  • View the files in Cloud9, click `Ctrl-S` to save each file.
  • Then view your page and refresh your browser using `Ctrl-f5` on windows or `CMD + SHIFT + R` on mac.
Possible Reason #3

You forgot to link your CSS file to your html file.

How to fix it

How to fix description

  • In the head of your html file, type `link` and select “link snippet” from the dropdown that appears as you type.
  • In the href=”” attribute, type the path to your CSS file between the two quotation marks.

(See below for how to find the right file path)
